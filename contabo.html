Compatible with Amazon S3 Standard






Our S3 API is structured the same way as AWS’s S3 API, so you can use the same code to connect to Amazon S3, Contabo Object Storage, or any other object storage provider that follows the S3 standard. We also offer standard S3 object storage features like versioning and locking.
Redundancy and End-to-End Encryption






We ensure redundancy of your data with triple-replication data protection, and all of your data is encrypted end-to-end. And when it comes to the physical security of your cloud object storage instance at our data centers, we make sure everything is under 24/7 monitoring with RBAC and biometric security measures.
High Performance and Speed






Our object storage has high-speed throughput (as fast as most fiber optic Internet connections) with as low latency as possible regardless of where you’re located. And if you’re curious about the hardware configurations, know that we followed all of RedHat’s recommendations for Ceph clusters word for word.
Easy Migration






Migrating data to Contabo Object Storage is easy. You can use S3-compatible API or Web UI to move your data. Changing your existing code can also be done in minutes. All you have to do is change the credentials and endpoint in your existing application.
Object Storage Use Cases
File Sharing
ACL management in Contabo Object Storage offers file-level granularity so you can determine who gets read or write access, and whether the URL is public or private.
Backup Files
Most customers use our object storage for file backups. You can integrate various backup tools with Contabo Object Storage in a few clicks.
Large Objects
You can store objects of all sizes, including large files up to 5TB each. Our customers typically store multimedia files (e.g. videos, photos, and audio), copies of operating system images, and database dumps.
Cold Storage
Many customers also use our object storage for cheap, long-term storage of data that doesn’t change often or that they don’t need to access frequently.
Storing Logs
Logs are often another object that our customers often store in Contabo Object Storage (most store their application logs in cold storage for retention and retrieval in future security investigations).
NAS Alternative
You can use object storage as a NAS alternative and let your cloud instances read and write data as needed.
